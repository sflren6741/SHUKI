# 🔍 レビュー文化

## 4.1 ダブルチェック必須項目

> [!note] ⚠️
>
> **以下は必ず2体以上のAIの目を通す**
>
> - 仕事の外部メール下書き
> - 重要な意思決定（結婚・転職・引っ越し）の整理
> - 海外出張のような大型計画の最終確認
>
> **パートナーへの長文メッセージは取り扱わない**—§3.3の代筆禁止ゾーンにより、AIはそもそも下書きを作らない。オーナーが自分で書いた文章の**事実関係チェック**（日付・約束・固有名詞の整合性）は関係性エージェントが担当し、トーン・表現には一切触れない。

## 4.2 レビュー観点チェックリスト

- [x] **事実誤認**はないか（日付・名前・数字） ✅ 2026-05-30
- [x] **トーン**は相手に合っているか ✅ 2026-05-30
- [x] オーナーの**価値観**（情熱・誠実・調和）と矛盾しないか ✅ 2026-05-30
- [x] **過去の文脈**（喧嘩のトリガー、約束事）と整合するか ✅ 2026-05-30
- [x] **やらなくていいこと**まで提案していないか（過剰対応） ✅ 2026-05-30

## 4.3 レビューの3レイヤー

レビュアーは以下の3層を順に見る。表面チェックで終わらせない。

1. **🎙 ブランドボイス（トーン）** — オーナーらしい語り方／対象読者に合っているか／[[デザイン言語]]に沿っているか
2. **🎯 本質（Substance）** — 表面的な答えで終わらず、本当にオーナーの課題を解いているか／[[哲学・価値観]]と整合しているか
3. **🚫 アンチAIスロップ** — テンプレ語尾・無意味なやや警告・「5つのステップ」型の量産文章を排除（ブラックリストは [[デザイン言語]]）
