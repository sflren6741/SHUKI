ここは **Claude Code と専門エージェント全員が参照するイントラネット** です。
オーナー（人間の意思決定者）がルールを書き、各AIはここを読んで動きます。

## 🧠 Local AI（Ollama）— 補助レイヤー

Claude Code とは別レイヤーで動く。`<LOCAL_AI_DIR>\` で管理。

- `local_qa_loop.py` — 毎日 00:01-18:00 連続ループ、vault の誤字脱字・frontmatter不整合を検出（18:00-24:00 はゴールデンタイム停止）
- 出力先：`00_Intranet/local-ai-runs/qa/YYYY-MM-DD.md`
- **vault 本体への書き込み禁止**。検出レポートだけ追記する設計

朝の orchestrator がレポートを読み、人間に提示する。

## 🤖 動き方の原則

- **自分の担当ページだけ読む**。イントラ全文を読み込まない（ハルシネーション防止）
- **タスクは `02_Tasks/` を参照**（自律ピック条件は ✅ タスク管理ルール）
- **ハンドオフは外に書く**（タスクボード or 共有ドキュメント。チャット内で済ませない）
- **役割の境界線を守る**（何をしないかを明示）
