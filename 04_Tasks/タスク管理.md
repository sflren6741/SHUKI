# ✅ タスク管理

> 1ファイル1タスク（`タスク管理/タスク/` 配下）。`ai_judgment_ok: true` かつ `completion_criteria` 記入済みのタスクはエージェントが自律実行。

```base
filters:
  and:
    - file.inFolder("04_Tasks/タスク管理/タスク")
    - status != "archived"
views:
  - type: table
    name: 🗂 アクティブ
    filters:
      and:
        - status.containsAny("todo", "in-progress")
    groupBy:
      property: status
      direction: ASC
    order:
      - file.name
      - status
      - priority
      - due
      - area
      - agent
    sort:
      - property: priority
        direction: DESC
      - property: due
        direction: ASC
  - type: table
    name: 📅 今週・期限
    filters:
      and:
        - due <= date(today, 7)
        - status != "done"
        - status != "cancelled"
    order:
      - file.name
      - status
      - priority
      - due
      - area
    sort:
      - property: due
        direction: ASC
      - property: priority
        direction: DESC
  - type: table
    name: 🤖 エージェント別
    filters:
      and:
        - status.containsAny("todo", "in-progress", "on-hold")
    groupBy:
      property: agent
      direction: ASC
    order:
      - file.name
      - status
      - priority
      - due
      - ai_judgment_ok
    sort:
      - property: priority
        direction: DESC
      - property: due
        direction: ASC
  - type: table
    name: 📋 全件
    order:
      - file.name
      - status
      - priority
      - due
      - area
      - agent
    sort:
      - property: status
        direction: ASC
      - property: due
        direction: ASC
```
