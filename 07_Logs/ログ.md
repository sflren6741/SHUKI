# 📓 ログ

> 日付つきの時系列記録。スタンドアップ・週次レビュー・内省・MTGメモ・分析など。  
> 「Log か Resource か」の判断: **日付の固有性で意味が決まるか？** Yes → Log。No → `06_Resources/`。  
> 重要なログには `## 気づき` + 概念ノードへのリンクを追加してグラフを編む。

```base
filters:
  and:
    - file.inFolder("07_Logs")
    - status != "archived"
views:
  - type: table
    name: 📅 最近のログ
    filters:
      and:
        - file.inFolder("07_Logs/ログ")
    groupBy:
      property: area
      direction: ASC
    order:
      - file.name
      - area
      - date
      - person
    sort:
      - property: date
        direction: DESC
  - type: table
    name: ☀️ スタンドアップ
    filters:
      and:
        - file.inFolder("07_Logs/朝スタンドアップ")
    order:
      - file.name
      - date
    sort:
      - property: date
        direction: DESC
  - type: table
    name: 📋 全件
    order:
      - file.name
      - date
    sort:
      - property: date
        direction: DESC
```
