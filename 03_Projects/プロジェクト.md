# 🗂 プロジェクト

> 期限つきで、複数のタスク・ログ・ナレッジが束になって初めて意味を持つ取り組み（`type: project`）。  
> 完了したら `status: done` → 後で `status: archived`。物理移動しない。

```base
filters:
  and:
    - file.inFolder("03_Projects/プロジェクト")
    - status != "archived"
views:
  - type: table
    name: 🚀 アクティブ
    filters:
      and:
        - status.containsAny("todo", "in-progress", "on-hold")
    groupBy:
      property: status
      direction: ASC
    order:
      - file.name
      - status
      - area
      - date
    sort:
      - property: date
        direction: ASC
  - type: table
    name: 📋 全件
    order:
      - file.name
      - status
      - area
      - date
    sort:
      - property: status
        direction: ASC
      - property: date
        direction: ASC
```
